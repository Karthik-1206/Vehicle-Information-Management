Vehicle Information Management is a open source website,everyone can access at any place.It consists of basics informations of the vehicles stored in the database. Anyone can access the details of the vehicles using the vehicle number of anyone's which is in active status.

Vehicle Information Management is proposed with the intention of upgrading the current system to web-based or online system. The basic information of the vehicle can be given by user itself. The register and login process is too easy. The information and all data given by user is store in the database.

The main motive of this website is to view vehicles information. This website contains the basic information of vehicles. This website is secure. The vehicles information is retrieved from the database.

Scope

To register and keep the information about all vehicles. Report from the database records in order to get the whole view about the vehicles information. A web based tool where user can search the information of the vehicle.
